px (pixels)

px is a fixed-size unit. It does not scale with screen size, zoom level, or user settings. Useful for pixel-perfect elements like icons, borders, or images where you want precise control.

Use the control buttons above to change the root font size — this box(200px × 60px) won’t change at all.

rem (root-em)

12rem × 4rem

rem is relative to the root element’s font size (usually 16px). When you change the root font size, rem-based values scale automatically. Great for responsive layouts and accessibility.

Use the control buttons above to change the root font size — this box(12rem × 4rem) will grow or shrink accordingly.
